# Bank Management System (Streamlit)

A simple banking system built using Python and Streamlit.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- View Account Details
- Delete Account

## Tech Stack
- Python
- Streamlit
- JSON (for storage)

## Run Locally

```bash
pip install streamlit
streamlit run app.py
```
main.py is basic python file which run locally on your terminal
